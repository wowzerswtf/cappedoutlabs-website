# Capped Out Labs — brand assets

Generated from brand tokens (navy `#1a3a5c`, electric `#2563EB`, Inter + JetBrains Mono).
Editable sources live in `/brand-assets/*.html` (render with `brand-assets/render.py` +
`brand-assets/resize.mjs`). Logomark concept: "breaking the cap" — ascending bars
punching through a ceiling line.

Live URLs once deployed: `https://cappedoutlabs.com/brand/<file>`

| File | Dimensions | Use |
|------|-----------|-----|
| `avatar-400.png` | 400×400 | LinkedIn company logo, X profile photo |
| `avatar-512.png` | 512×512 | Crunchbase logo (and any larger square need) |
| `linkedin-banner.png` | 1128×191 | LinkedIn company page banner |
| `x-header.png` | 1500×500 | X / Twitter header |
| `logo-horizontal-dark.png` | 1200×360, transparent | Wordmark for light backgrounds (decks, site, docs) |
| `logo-horizontal-light.png` | 1200×360, transparent | Wordmark for dark backgrounds |

## Per-platform upload

- **LinkedIn:** logo → `avatar-400.png`; banner → `linkedin-banner.png` (the company
  logo overlaps the lower-left, which the banner keeps clear).
- **X / Twitter:** profile photo → `avatar-400.png`; header → `x-header.png` (avatar
  overlaps lower-left; key content is centered).
- **Crunchbase:** logo → `avatar-512.png`.
